Enchantment Descriptions
========================
1. Enchantments Control can use descriptions provided for ED.
 - Enchantments Control will use enchantment description from any mod, which provides 
 ED descriptions (only if EC config doesn't contain description for enchantment).
2. ED can use descriptions provided by EC.
 - If EC config contains descriptions for some enchantments it will be used by 
 ED (only if there is no description, provided for ED).
